# Example Ultravox client
This demonstrates a simple example of using the Ultravox client in Python.

Run from the example directory with:

```
uv run python ultravox_example/client.py --join-url=$JOIN_URL
```

You can get a `join_url` from the [Ultravox REST API](https://docs.ultravox.ai/api-reference/agents/agents-calls-post).
